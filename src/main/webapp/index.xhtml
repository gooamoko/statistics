Сервер статистики

Как пользоваться сервером статистики

Пользоваться данным сервером просто. Для получения требуемого отчета вам необходимо выполнить три действия:

  • Выбрать отчет из выпадающего списка;
  • Уточнить параметры отчета, которые появятся снизу под выпадающим списком;
  • Нажать на кнопку "Отобразить".

Если отчет не появился, либо он пуст, то это возможно по следующим причинам:

  • Для отображения списка зачисленных студентов и количества поданных заявлений отчеты перестают возвращать данные после наступления нового календарного года. Такова логика работы приложения;
  • Для других отчетов данные должны возвращаться всегда. Если данные отсутствуют, это может быть из-за проблем с подключением к базе данных.

Если данные по успеваемости и посещаемости отображаются, но в поле "Номер студенческого" ничего нет, это означает, что на отделении в базе данных отсутствует номер вашего студенческого. Обратитесь на отделение, чтобы его указали и вы сможете просматривать детали вашей успеваемости.
